Bước 1: Chuẩn bị Google Sheet

Tạo một Google Sheet mới. Tại dòng đầu tiên (hàng 1), nhập tên 4 cột lần lượt là: Timestamp, Tên, Tham dự, Thiệp.

Bước 2: Cấu hình Apps Script

Trong Sheet, chọn Tiện ích > Apps Script. Xóa toàn bộ code mặc định và dán đoạn code dưới đây vào: 

function doPost(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    new Date().toLocaleString('vi-VN'),
    data.ten,
    data.thamDu,
    data.thiep
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}
Bước 3: Triển khai (Deploy)

1. Nhấn nút Triển khai > Triển khai mới.
2. Chọn loại Ứng dụng web.
3. Cấu hình: Thực thi dưới quyền: Tôi; Ai có quyền truy cập: Bất kỳ ai.
4. Nhấn Triển khai. (Nếu hiện bảng yêu cầu, chọn Nâng cao > Đi tới dự án (không an toàn) > Cho phép).
5. Copy đường dẫn URL ứng dụng web và dán vào ô "Google Apps Script URL" trong trang này.
